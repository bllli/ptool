# ptool
发种工具ptool运行在发种员机器上

## 功能
- [ ] 用户输入
    - [ ] 选择本地文件/文件夹
    - [ ] 输入豆瓣地址
- [ ] 自动生成
    - [ ] 自动为选中的视频文件创建缩略图 `ffmpeg`
    - [ ] 自动上传缩略图到图床
    - [ ] 自动捕获视频文件的media info `mediainfo`
    - [ ] 捕获豆瓣电影信息、海报
    - [ ] 海报上传到图床
    - [ ] bbcode格式简介
- [ ] 最终产出
    - 在海胆pt站创建种子条目
    - 种子文件

## 依赖
- python3.7
- ffmpeg
- mediainfo
